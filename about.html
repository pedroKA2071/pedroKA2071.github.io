Sobre Mim

Olá, sou um desenvolvedor web apaixonado por carros e basquete, com formação em Tecnologia da Informação. Minha jornada no mundo da tecnologia começou quando decidi mergulhar no universo da programação. Com o tempo, adquiri uma sólida base de conhecimento e habilidades que me permitiram criar soluções web incríveis.

Minhas habilidades incluem HTML, CSS e JavaScript, que formam a base do desenvolvimento web. Além disso, domino o Angular, uma estrutura poderosa para a criação de aplicativos web dinâmicos. Para gerenciar e colaborar em projetos, sou experiente em Git e GitHub, garantindo um fluxo de trabalho eficiente e colaborativo Vamos criar algo incrível juntos!
